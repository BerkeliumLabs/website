# Install

To begin, you can install berkelium using npm or yarn:

npm i berkelium

or

yarn add berkelium

# Import berkelium

Simply import berkelium to your project as follows;

For ESModule syntax

import { bkLabs } from 'berkelium';

or CommonJS syntax

const { bkLabs } = require('berkelium');
